Welcome to Gaitoraid

Gaitoraid is an open source image processing algorithm developed mainly for use in traffic cameras and identifying people through their gait information.

Fork our repo and contribute to it.

$ git clone https://github.com/sgang007/gaitoraid

Approaches to be tried

We have two approaches in mind. 1. Variational Approximation on Bayesian Inference to match the model with the existing database. 2. Usage of ANNs to do Deep Learning on training images and learning gait information.
